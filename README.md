## Overview 🚀

Learning Atlas guides you on the right path to learn anything! The user types in what they intend to learn, and the API (OpenAI) recommends high-quality resources with a description and an option to bookmark them.

**Features/Highlights**: 📌
- Responsive design
- Fetches resources quickly  🔺 _(note: the database is still small! Working on expanding it!)_ 🔺 
- Ability to bookmark resources and quick access to bookmark sections
